## Mad Libs
#### Play this in your code editor terminal. My favorite editor is VS Code<hr>
A Mad Libs game asks the user for different kinds of words (like nouns, adjectives, or places) and fills them into a story.

For example:
If the user gives these words:

- Noun: ant

- Adjective: sad

- Place: forest

The game might create a sentence like:
“The forest can be a sad place for an ant when it rains.”
