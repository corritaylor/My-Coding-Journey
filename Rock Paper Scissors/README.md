## Rock, Paper, Scissors
#### Play this in your code editor terminal. My favorite editor is VS Code<hr>

By the end, you'll have a fun Rock, Paper, Scissors game where you play against the computer and keep score.

This easy project will teach you basic programming skills like:

- Making the computer choose randomly  
- Checking if the player's input is valid  
- Using loops and if-statements to control the game  
- Keeping score and letting the player play again
