### Rock Paper Scissors<hr>
This tutorial will guide you through creating a Rock Paper Scissors game using Python!

By the end, you’ll have a working game where the user plays against the computer in an interactive, score-tracking version of Rock Paper Scissors.

This beginner-friendly project covers essential programming concepts, including:

- Using the random module to simulate a computer opponent<br>
- Handling user input validation using regular expressions<br>
- Working with loops and conditional statements<br>
- Keeping track of the score and allowing replay
